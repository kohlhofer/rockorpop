# Rock or Pop - Music Consultancy Website

A static website for Rock or Pop, a music consultancy business. This site is designed to be hosted on GitHub Pages.

## Setup Instructions

1. Add your background image:
   - Place your background image in the root directory
   - Name it `background.jpg`
   - Recommended size: 1920x1080 pixels or larger

2. Deploy to GitHub Pages:
   - Create a new repository on GitHub
   - Push these files to your repository
   - Go to repository Settings > Pages
   - Select the main branch as the source
   - Your site will be published at `https://[your-username].github.io/[repository-name]`

3. Custom Domain Setup (Optional):
   - Purchase the domain rockorpop.com
   - In your repository settings, under GitHub Pages:
     - Add your custom domain
     - Enable HTTPS
   - Update your domain's DNS settings to point to GitHub Pages

## Local Development

To view the site locally:
1. Clone this repository
2. Open `index.html` in your web browser

## File Structure

- `index.html` - Main HTML file
- `styles.css` - CSS styles
- `background.jpg` - Background image (to be added)
- `README.md` - This file
